Einstellungen

Trage zuerst deinen Claude API Key ein (für die Buchanalyse), und optional den eBay Token für den Direktupload.

Claude API Key holen:
1. console.anthropic.com öffnen
2. „API Keys" → „Create Key" klicken
3. Key kopieren (beginnt mit sk-ant-...)
Für die KI-Buchanalyse · nur lokal gespeichert, nie übertragen
eBay (optional, für Direktupload):
1. developer.ebay.com → kostenlos registrieren
2. „Application Keys" → neue App anlegen
3. „User Tokens" → Token generieren und kopieren
Aus Developer Portal → User Tokens → OAuth User Token
Aus Developer Portal → Application Keys → App ID (Production)

Token wird nur lokal in deinem Browser gespeichert und nie an Dritte übertragen.
